Python OOP — Abstract Classes, Interfaces & Subclassing

1. Abstract Classes
Understand and apply abstract classes to define common interfaces while enforcing required methods and structure.

2. Interfaces & Duck Typing
Grasp how interfaces and duck typing ensure objects follow a specific contract or protocol through behavior rather than inheritance.

3. Subclassing Standard Base Classes
Learn to extend built‑in types (lists, dictionaries, iterators, etc.) to create custom data structures with specialized behavior.

4. Method Overriding
Use method overriding to modify or enhance inherited behavior from base classes.

5. Multiple Inheritance
Understand and apply multiple inheritance to form complex class relationships and shared behaviors.

6. Mixins
Utilize mixins to compose reusable behavior across unrelated classes without forming deep inheritance chains.
